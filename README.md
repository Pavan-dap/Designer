# React + Vite + AntD + Bootstrap + Kanban

A starter project with:

- React 18 + Vite 5
- Ant Design (UI components)
- Bootstrap (utility classes)
- Login screen (AntD form + Bootstrap styling)
- Kanban board with drag-and-drop (`@hello-pangea/dnd`)

## Quick Start

```bash
npm install
npm run dev
```
